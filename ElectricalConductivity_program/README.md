# Conductivity_program

Predicting electrical conductivity (sigma) of ionic liquids from molecular structure (SMILES), using Graph Neural Networks and traditional machine learning models with explainability analysis.

## Project Structure

```
├── configs/                  # YAML configuration files
├── data/
│   ├── raw/                  # Raw CSV data (SMILES, temperature, targets)
│   └── processed/            # Preprocessed features (graphs, fingerprints, etc.)
├── results/
│   ├── checkpoints/          # Saved model weights (.pth)
│   ├── figures/              # Plots (predictions, SHAP, error distributions)
│   └── fragment_explain/     # Fragment & node feature importance
├── scripts/                  # Entry-point scripts
│   ├── preprocess_data.py
│   ├── preprocess_graph_data.py
│   ├── preprocess_fragment_data.py
│   ├── run_gnn.py
│   ├── run_ml.py
│   ├── run_lazy_predict.py
│   ├── explain_single.py
│   ├── explain_whole.py
│   └── fragment_explain.py
└── src/
    ├── data_processing/      # SMILES → graphs / fingerprints / descriptors
    ├── datasets/             # PyTorch Dataset classes
    ├── models/               # GCN, GAT, GIN, MLP, RandomForest, XGBoost
    ├── training/             # Training loops
    ├── evaluation/           # Metrics, visualization, SHAP, GNNExplainer
    └── utils/                # Data loading, splitting
```

## Installation

**1. Install PyTorch (CUDA 13.2):**

```bash
pip3 install torch==2.12.0 torchvision==0.27.0 --index-url https://download.pytorch.org/whl/cu132
```

For CPU only:

```bash
pip3 install torch==2.12.0 torchvision==0.27.0
```

**2. Install remaining dependencies:**

```bash
pip install -r requirements.txt
```

## Usage

### Preprocessing

Convert raw CSV data into different feature representations:

```bash
# Molecular graphs (for GNN models)
python scripts/preprocess_graph_data.py

# Fragment-annotated graphs (for explainability)
python scripts/preprocess_fragment_data.py
```

Fingerprints, group counts, and molecular descriptors are generated by running the corresponding modules directly (e.g., `python -m src.data_processing.smiles2fingerprint`).

### Training & Evaluation

**GNN models** (GCN / GAT / GIN):

```bash
python scripts/run_gnn.py configs/gcn.yaml
python scripts/run_gnn.py configs/gat.yaml
python scripts/run_gnn.py configs/gin.yaml
```

**ML models** (MLP / RandomForest / XGBoost):

```bash
python scripts/run_ml.py configs/mlp_fp.yaml
python scripts/run_ml.py configs/rf_descriptor.yaml
python scripts/run_ml.py configs/xgboost_gc.yaml
```

### Explainability

```bash
# Fragment importance across all samples
python scripts/explain_whole.py

# Single sample explanation (atom importance heatmap)
python scripts/explain_single.py

# Visualize aggregate importance scores
python scripts/fragment_explain.py
```

## Configuration

All training parameters are specified in YAML config files under `configs/`. Key fields:

| Field | Description |
|---|---|
| `model.name` | Model type: `gcn`, `gat`, `gin`, `mlp`, `random_forest`, `xgboost` |
| `model.params` | Architecture / hyperparameter search space |
| `data.feature_type` | Feature type: `graph`, `fingerprint`, `group_count`, `descriptor` |
| `data.target` | Prediction target: `sigma` |
| `training.epochs` | Number of training epochs (deep learning models) |
| `training.lr` | Learning rate |
| `training.batch_size` | Batch size |
| `training.test_size` | Test set ratio (default 0.2) |

## Models

| Model | Type | Feature Input |
|---|---|---|
| GCN | Graph Neural Network | Molecular graph |
| GAT | Graph Neural Network | Molecular graph |
| GIN | Graph Neural Network | Molecular graph |
| MLP | Feedforward Network | Fingerprint / group count / descriptors |
| Random Forest | Ensemble | Fingerprint / group count / descriptors |
| XGBoost | Gradient Boosting | Fingerprint / group count / descriptors |

## Data Format

Raw data (`data/raw/whole.csv`):

| Column | Description |
|---|---|
| `IL` | Ionic liquid ID |
| `IL_smiles` | SMILES string (cation.anion) |
| `Cation` | Cation SMILES |
| `Anion` | Anion SMILES |
| `T` | Temperature (K) |
| `sigma` | Target electrical conductivity |
